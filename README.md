# Student Performance & Result Analytics
Naan Mudhalvan Arts Internship 2026 - Python for Data Analytics

## Problem Statement
Colleges manually analyze student marks, making it difficult to identify weak students quickly. This project automates grade calculation and visualizes performance.

## Objectives
1. Calculate Total, Average, Grade, Pass/Fail automatically
2. Identify Toppers and Weak Students  
3. Subject-wise & Class-wise performance analysis
4. Generate visual dashboard for decision-making

## Technologies Used
- **Language**: Python 3
- **Environment**: Google Colab  
- **Libraries**: Pandas, NumPy, Matplotlib, Seaborn

## Dataset
`students_marks.csv` contains 8 students with marks in Tamil, English, Maths, Science, Social.

## Key Features
- Automated grading system with A+ to F grades
- Pass/Fail status based on 35 marks criteria
- Interactive dashboard with 4 visualizations
- CSV export of final results

## Output Files
1. `final_student_result.csv` - Result with Grade
2. `performance_dashboard.png` - 4 Charts Dashboard
